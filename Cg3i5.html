알고리즘

 프로그램 시작 
    먼저 맵 출력 
    캐릭터를 기존 위치에서 지우고 이동한 새 위치에 출력 
    게임 종료 확인 
    게임 종료 아니면 새 이동 방향 입력 받아서 게임 계속 진행 
 프로그램 종료 
 
 맵 출력 함수 
    길, 벽, 옷, 문의 출력값 설정 
    2차원 미로 맵을 한 칸씩 출력 
 
 맵 이동 함수 
    W는 길 위쪽으로 한칸 이동 
    A는 길 왼쪽으로 한칸 이동 
    S는 길 아래쪽으로 한칸 이동 
    D는 길 오른쪽으로 한칸 이동 
 
 게임 종료 확인 함수 
    목적지에 도착하면 종료 
    아니면 게임 계속 진행
