문제

char, int, double 형 변수에 'A', 7, 3.141592의 값을 저장하고 포인터 변수에서 각 변수의 주소값을 
저장하고 기본 자료형 변수와 포인터 변수로 주소값 및 저장값 출력하는 문제입니다 이를 해결하는 다음 프로그램의 실행상태를 확인하세요 
변수명( 주소값 )  저장값   포인터변수가 가리키는 값
-------------------------------------------------
var1( 0018FF44 )  A
ptr1( 0018FF34 )  0018FF44  A
var2( 0018FF40 )  7
ptr2( 0018FF30 )  0018FF40  7
var3( 0018FF38 )  3.141592
ptr3( 0018FF2C )  0018FF38  3.141592

1과 4 5 사이의 실행순서(2개) 및 실행상태

2  double var3 = 3.141592;
main()                  
var1 (0018FF44번지)
var2 (0018FF40번지)
var3 (0018FF38번지)
ptr1 (0018FF34번지)
ptr2 (0018FF30번지)
ptr3 (0018FF2C번지)
 

3  double *ptr3 = &var3;
main()                  
var1 'A' (0018FF44번지)
var2 7 (0018FF40번지)
var3 3.141592 (0018FF38번지)
ptr1 (0018FF34번지)
ptr2 (0018FF30번지)
ptr3 (0018FF2C번지)


프로그램 코드

	
	#include <stdio.h>
		
1	int main() {					
		char var1 = 'A';	
		int var2 = 7;	
2		double var3 = 3.141592;
		
		char *ptr1 = &var1;				
		int *ptr2 = &var2;				
3		double *ptr3 = &var3;				
	
		printf( "변수명(주소값)   저장값   포인터변수가 가리키는 값\n-------------------------\n" );
		printf( "var1(%p)  %c    \n", &var1, var1 );			
4		printf( "ptr1(%p)  %p  %c\n", &ptr1, ptr1, *ptr1 );				
		printf( "var2(%p)  %d    \n", &var2, var2 );			
5		printf( "ptr2(%p)  %p  %d\n", &ptr2, ptr2, *ptr2 );				
		printf( "var3(%p)  %f    \n", &var3, var3 );			
6		printf( "ptr3(%p)  %p  %f\n", &ptr3, ptr3, *ptr3 );				
	
7		return 0;
	}
