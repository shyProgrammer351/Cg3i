문제

char, int, double 형 변수에 'A', 7, 3.141592의 값을 저장하고 포인터 변수에서 각 변수의 주소값을 
저장하고 기본 자료형 변수와 포인터 변수로 주소값 및 저장값 출력하는 문제입니다 이를 해결하는 다음 프로그램에 대해 빈칸을 채우세요 
변수명( 주소값 )  저장값   포인터변수가 가리키는 값
-------------------------------------------------
var1( 0018FF44 )  A
ptr1( 0018FF34 )  0018FF44  A
var2( 0018FF40 )  7
ptr2( 0018FF30 )  0018FF40  7
var3( 0018FF38 )  3.141592
ptr3( 0018FF2C )  0018FF38  3.141592

코드 빈칸 채우기

#include <stdio.h> 
	 
int main() { 
	 var1 = 'A';	 
	 var2 = 7;	 
	 var3 = 3.141592; 
	 
	ptr1 = var1;				 
	ptr2 = var2;				 
	ptr3 = var3;				 
 
	printf( "변수명(주소값)   저장값   포인터변수가 가리키는 값\n-------------------------\n" ); 
	printf( "var1()  %c    \n", var1, var1 );			 
	printf( "ptr1()    %c\n", ptr1, ptr1, ptr1 );				 
	printf( "var2()  %d    \n", var2, var2 );			 
	printf( "ptr2()    %d\n", ptr2, ptr2, ptr2 );				 
	printf( "var3()  %f    \n", var3, var3 );			 
	printf( "ptr3()    %f\n", ptr3, ptr3, ptr3 );				 
 
	return 0; 
}
