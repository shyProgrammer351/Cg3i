문제

우리반 학생들의 평균 점수를 계산하는 문제입니다 이를 해결하는 다음 프로그램의 실행상태를 확인하세요 
우리반 학생수를 입력하세요 : 3
점수를 입력하세요 : 90
점수를 입력하세요 : 89
점수를 입력하세요 : 86
우리반 학생들의 평균 점수는 88.33입니다

프로그램 코드

	#include <stdio.h>
	#include <stdlib.h>
		
1	int main() {
		int count = 0, index = 0, sum = 0;
2		int* scores = NULL;
	
		printf( "우리반 학생수를 입력하세요 : ");
3		scanf( "%d", &count );
	
4		scores = 
5		         (int*)malloc( sizeof(int) * count );
6		for( index = 0; index < count; index++ ) {
			printf( "점수를 입력하세요 : ");
7			scanf( "%d", &scores[index] );
		}
		
		for( index = 0; index < count; index++ ) {
			sum += scores[index];
		}
		
8		printf( "우리반 학생들의 평균 점수는 %.2f입니다", (double)sum / count );
		
9		free( scores );
		
10		return 0;
	}
