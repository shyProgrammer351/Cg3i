문제

학생의 이름, 국어 점수, 영어 점수, 수학 점수를 입력받아 평균을 계산하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요 
학생 이름을 입력하세요: 홍길동
0점 ~ 100점 사이의 국어 점수를 입력하세요: 90
0점 ~ 100점 사이의 영어 점수를 입력하세요: 89
0점 ~ 100점 사이의 수학 점수를 입력하세요: 86
홍길동 학생은 국어 90점, 영어 89점, 수학 86점, 평균 88.33점입니다


학생 이름을 입력하세요: 주먹이
0점 ~ 100점 사이의 국어 점수를 입력하세요: 76
0점 ~ 100점 사이의 영어 점수를 입력하세요: 95
0점 ~ 100점 사이의 수학 점수를 입력하세요: 83
주먹이 학생은 국어 76점, 영어 95점, 수학 83점, 평균 84.67점입니다


학생 이름을 입력하세요: 반쪽이
0점 ~ 100점 사이의 국어 점수를 입력하세요: 97
0점 ~ 100점 사이의 영어 점수를 입력하세요: 93
0점 ~ 100점 사이의 수학 점수를 입력하세요: 91
반쪽이 학생은 국어 97점, 영어 93점, 수학 91점, 평균 93.67점입니다

   학생 정보를 초기화 
   학생 정보를 입력 및 계산

프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter13/Student.c

	#include <stdio.h>

	 

	// 학생 정보 구조체 정의

	typedef struct _Student{

		char name[16];

		int korean;

		int english;

		int mathematics;

		double average;

	} Student;

	 

	// 프로그램시작 

1	int main() {

		// 학생 정보를 초기화  

2		Student student = { "", 0, 0, 0, 0.0 };

	 

		// 학생 정보를 입력 및 계산  

		printf( "학생 이름을 입력하세요: " );

		scanf( "%s", student.name );

		printf( "0점 ~ 100점 사이의 국어 점수를 입력하세요: " );

		scanf( "%d", &student.korean );

		printf( "0점 ~ 100점 사이의 영어 점수를 입력하세요: " );

		scanf( "%d", &student.english );

		printf( "0점 ~ 100점 사이의 수학 점수를 입력하세요: " );

		scanf( "%d", &student.mathematics );

3		student.average = (double)( student.korean + student.english + student.mathematics ) / 3;

	 

		// 학생 정보를 출력 

4		printf( "%s 학생은 국어 %d점, 영어 %d점, 수학 %d점, 평균 %.2f점입니다",

			student.name, student.korean, student.english, student.mathematics, student.average );

		 

		// 프로그램 종료 

5		return 0;

	}
